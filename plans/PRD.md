# WeaveFFI — Product Requirements Document

WeaveFFI is a Rust CLI that generates multi-language FFI bindings from a single
API definition file (YAML/JSON/TOML). It targets C, Swift, Android/Kotlin,
Node.js, and WASM. The generated code calls into a user-written Rust library
through a stable C ABI.

**Current state:** The CLI works for flat functions with primitive types (i32,
u32, i64, f64, bool, string, bytes, handle). It has 5 generators, a calculator
sample, and 15 tests. It does NOT support structs, enums, optional types, or
arrays — making it too limited for real-world use.

**Goal:** Make WeaveFFI feature-complete enough for a stable release on
crates.io. That means supporting complex types (structs, enums, optionals,
lists), having comprehensive tests, complete documentation, and a non-trivial
sample that demonstrates all features.
